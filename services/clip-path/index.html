CSS clip-path 產生器

← 回工具箱

拖曳多邊形頂點調出裁切形狀，即時預覽裁切結果，一鍵複製可直接貼用的 clip-path: polygon()。內建三角、箭頭、星形、對話框等常用形狀，全程在瀏覽器端運算。

預設形狀 · presets
頂點座標 · points 3 點

單位＝%（0–100）。拖曳 ● 移動頂點、點邊線中點 ＋ 插入頂點、雙擊 ● 刪除（至少保留 3 點）；選中頂點可用方向鍵微調（Shift ＝ 大步）。

預覽比例 · aspect

百分比座標會跟著容器比例變形，換比例可先確認形狀在目標容器上的模樣。
